== VPStreamer for Mac (aka "Vidpresso Encoding Helper")
Copyright (c) Vidpresso Inc. 2017-18


This is a macOS command-line encoding and streaming tool using the libobs library.

An Xcode project file is provided. Only the target "Vidpresso Encoding Helper" is relevant for this tool. 

Prebuilt libobs+ffmpeg static library binaries are provided in directories: ObsLibs, ObsData, ffmpeglibs
